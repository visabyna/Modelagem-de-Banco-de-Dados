Modelagem de Banco de Dados

Repositório dedicado aos exercícios, atividades e projetos da disciplina Modelagem de Banco de Dados, cursada no 1º semestre da faculdade de Ciência da Computação.

Aqui registro tudo que aprendi sobre como estruturar, organizar e representar informações de forma lógica e eficiente.

 Sobre a Disciplina

A disciplina de Modelagem de Banco de Dados ensina como analisar, organizar e representar dados para que sistemas possam armazená-los e utilizá-los corretamente.
O foco inicial é aprender a criar modelos que representem o mundo real e entendê-los antes de implementar um banco de dados em si.
